# spherix public bugtracker
Публичный багтрекер для более удобного отслеживания багов и проблем в сервисах sphrix.cc
За активную помощь в обнаружении багов - поощряем (финансово/внутриигровыми плюшками)

Просьба заполнять репорты информативно, не кратко, так легче разобраться в проблеме
[Создать репорт [клик]](https://github.com/sphrixcc/sphrix-public-bugtracker/issues/new/choose)
#### Виды репортов
- Minecraft bug - проблема на *майнкрафт* сервере. Например, баг в работе плагинов
- Frontend bug - проблема на стороне фронтэнда сайта. Например, визуальный баг отображения сообществ
- API bug - проблема на стороне бекенда сайта. Например, при смене пароля произошла ошибка и пароль неверно сменился
- Предложение по серверу
